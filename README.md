# React-Native-Education-App
## Introduction
The React Native Education App is a mobile application designed for learning and institution management. It provides a user-friendly interface for students, teachers, and administrators to manage their educational activities.
## Key Features
- Institution and teacher profiles
- Course and subject management
- Search and filter functionality
- User authentication and authorization
## Tech Stack
- React Native
- Expo
- NativeWind
- React Native Heroicons
## Installation
To install the project, run the following commands:
```\nnpm install\n```\n## Usage
To start the application, run the following command:
```\nexpo start\n```\n## Environment Variables
No environment variables are required for this project.
## Project Structure
The project is structured into the following components:
- App.js: The main application component
- components: A directory containing various reusable components
- navigation: A directory containing the application navigation
- theme: A directory containing the application theme and styles
- assets: A directory containing images and other assets
